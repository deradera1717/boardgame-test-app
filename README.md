# Next.js + pnpm + Vercel Starter

TypeScript・App Router を使った Next.js の最小構成テンプレート。pnpm を使ってローカル開発し、Vercel にデプロイすることを想定しています。

## 前提

- Node.js 18+（.nvmrc は 20 を指定）
- pnpm 9+

## セットアップ

```bash
pnpm install
pnpm dev
```

## 推奨 Vercel 設定

- Install Command: `pnpm install`
- Build Command: `pnpm build`
- Environment Variables: 必要に応じて設定
